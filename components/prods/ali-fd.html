# 前端说明

统一安装步骤

  • npm install --registry=https://registry.npm.alibaba-inc.com -g tnpm
  • tnpm i -g @ali/def-pub-client
  • tnpm install @ali/def -g
  • git clone git@gitlab.alibaba-inc.com:aone-app-stack/frontend.git
  • cd frontend && tnpm install
  • tnpm start

# do your work

  • dps -d // 发布日常

or

  • dps -o // 发布线上
